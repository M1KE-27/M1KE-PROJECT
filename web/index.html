What's inside

🟠 Own kernel

32-bit hybrid kernel: GDT/IDT, IRQs, PIT, PS/2 keyboard & mouse, kernel heap, and a RAM filesystem. Boots via Multiboot/GRUB.

🪟 Glass desktop

A compositor written from scratch on the linear framebuffer: blur, translucency, rounded corners, soft shadows, taskbar & start menu.

🎛️ Control plane

m1kectl inspects and controls every subsystem: scheduler, memory, modules, services, windows — all from the terminal.

🎨 m1ss language

Our own CSS-like styling language. Edit /etc/m1ke/theme.m1ss, run m1kectl theme reload, and the whole UI restyles live.

📦 m1pkg

A package manager with a built-in repository that installs real files into the filesystem. m1pkg install <pkg>.

🧰 Real apps

A Terminal that runs the actual shell, a text Editor that saves, a Files browser, a System Monitor and a Settings panel.

🔌 Hot modules

Load/unload drivers at runtime: m1kectl module unload mouse really masks its IRQ. Nothing is sacred.

🔎 Observable

A live event bus (dmesg) and m1kectl inspect let you watch and query the running system.

Roadmap

  1. Fase 0 — Base: boot, drivers, heap, ramfs, shell, packages.
  2. Fase 1 — Control plane: m1kectl, config en texto, eventos.
  3. Fase 1.5 — Glass desktop + lenguaje m1ss + temas en vivo.
  4. Fase 2 — Scheduler preventivo + procesos reales.
  5. Fase 3 — Memoria virtual (paging).
  6. Fase 4 — Userspace + cargador ELF + syscalls.
  7. Fase 5 — IPC + módulos dinámicos en caliente.
  8. Fase 6 — VFS + disco persistente.
  9. Fase 7 — Compositor/protocolo gráfico propio.
  10. Fase 8 — Networking (TCP/IP). 9 Audio/USB. 10 SMP.

Download & build

m1keOS ships as source you compile into a bootable ISO. Released version by version.

latest: v0.5 "Naranja"
# Arch / CachyOS
sudo pacman -S nasm grub qemu

git clone https://github.com/<your-user>/m1ke-project
cd m1ke-project
make all      # builds m1keos.iso
make gui      # boots the graphical desktop
↓ m1keos.iso Source on GitHub

Open source. Builds anywhere with gcc -m32, nasm, grub and qemu.
